  136: ## Data Flow
  137: 
  138: ### CLI Session
  139: 
  140: ```text
  141: User input вҶ’ HermesCLI.process_input()
  142:   вҶ’ AIAgent.run_conversation()
  143:     вҶ’ prompt_builder.build_system_prompt()
  144:     вҶ’ runtime_provider.resolve_runtime_provider()
  145:     вҶ’ API call (chat_completions / codex_responses / anthropic_messages)
  146:     вҶ’ tool_calls? вҶ’ model_tools.handle_function_call() вҶ’ loop
  147:     вҶ’ final response вҶ’ display вҶ’ save to SessionDB
  148: ```
  149: 
  150: ### Gateway Message
  151: 
  152: ```text
  153: Platform event вҶ’ Adapter.on_message() вҶ’ MessageEvent
  154:   вҶ’ GatewayRunner._handle_message()
  155:     вҶ’ authorize user
  156:     вҶ’ resolve session key
  157:     вҶ’ create AIAgent with session history
  158:     вҶ’ AIAgent.run_conversation()
  159:     вҶ’ deliver response back through adapter
  160: ```
  161: 
  162: ### Cron Job
  163: 
  164: ```text
  165: Scheduler tick вҶ’ load due jobs from jobs.json
  166:   вҶ’ create fresh AIAgent (no history)
  167:   вҶ’ inject attached skills as context
  168:   вҶ’ run job prompt
  169:   вҶ’ deliver response to target platform
  170:   вҶ’ update job state and next_run
  171: ```
